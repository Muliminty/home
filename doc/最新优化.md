## 🎯 使用方法

### 启动项目
```bash
npm start
```

### 尝试这些命令
```
help        # 查看所有可用命令
about       # 查看个人简介
projects    # 浏览项目列表
blogs       # 查看技术博客
link        # 联系方式
history     # 命令历史
clear       # 清空屏幕
```

### 尝试这些快捷键
- 按住 `Ctrl` + 按 `L` = 清屏
- 按 `ESC` = 清空输入
- 在help中按 `↑` `↓` = 选择命令
- 按 `Enter` = 执行命令
