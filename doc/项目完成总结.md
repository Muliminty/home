## 🎯 用户体验

### 交互流程

```
用户访问 → 看到粒子背景 → ASCII欢迎 → 自动聚焦
    ↓
输入命令 → 实时执行 → 美观输出 → 保持聚焦
    ↓
blogs命令 → 显示列表 → 点击文章 → 渲染MDX
    ↓
滚动阅读 → 输入框固定 → 随时输入新命令
```

### 快捷操作

```
1. Ctrl+L → 清屏 → 输入 help → 查看帮助
2. 按 ↑ 浏览历史 → 选择命令 → Enter执行
3. blogs → 点击文章 → 阅读 → 继续输入
4. ESC → 清空输入 → 重新开始
```
